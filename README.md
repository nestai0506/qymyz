# Бал қымыз 🥛

Қазақы стильдегі, дәстүрлі қымыз сататын лендинг-сайт. Таза HTML, CSS және
vanilla JavaScript-те жасалған — ешқандай framework жоқ.

## Мүмкіндіктер

- 🎨 **Қазақы дизайн** — қошқар мүйіз ою-өрнегі, түнгі көк/қоңыр/сарғыш палитра
- 🛒 **Себет** — тауар қосу/өшіру, санын өзгерту, жалпы соманы автоматты есептеу
- 💬 **WhatsApp интеграциясы** — тапсырыс тізімі автоматты хабарлама болып жіберіледі
- 🌐 **3 тіл** — қазақша / орысша / ағылшынша (таңдау есте сақталады)
- 🌀 **Ultra 3D анимация** — торсықтан кесеге қымыз құйылу сахнасы (гироскоп + тінтуірмен бұрылады)
- 📱 **Толық адаптивті** — мобильді, планшет, десктоп
- 📸 **Instagram** сілтемесі: [@_estai_n](https://instagram.com/_estai_n)

## Файлдар

| Файл | Сипаты |
|------|--------|
| `index.html` | Беттің құрылымы |
| `styles.css` | Дизайн, анимация, адаптивтілік |
| `script.js` | Себет, WhatsApp, 3D сахна, аударма логикасы |
| `images/` | Суреттер (hero, about фондары) |
| `қымыз.html` | Барлығы біріктірілген жалғыз файл (телефонға/офлайн жіберуге ыңғайлы) |

## Іске қосу

`index.html` файлын браузерде ашсаңыз болғаны. Немесе VS Code-та **Live Server** қолданыңыз.

## Баптау

- WhatsApp нөмірі: `script.js` ішіндегі `WHATSAPP_PHONE`
- Өнімдер тізімі: `script.js` ішіндегі `PRODUCTS`
